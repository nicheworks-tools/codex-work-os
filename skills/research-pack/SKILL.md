# Research Pack

## Purpose
Turn raw notes and source fragments into clear comparison briefs and decision-ready memos.

## Use this pack when
- comparing options, vendors, competitors, or approaches
- turning messy notes into a structured recommendation
- summarizing multiple sources into one readable brief
- writing a decision memo with evidence and uncertainty
- clarifying trade-offs before choosing a path

## Do not use this pack when
- the main task is a PM status update
- the task is a customer escalation handoff
- the request is primarily an outward-facing sales reply
- the task is executive calendar or meeting support
- the user needs a code implementation rather than a research artifact

## Inputs
- notes, excerpts, links, internal documents, or source summaries
- decision question if known
- evaluation criteria if known
- business constraints and risks if known

## Required behavior
1. Make the decision question explicit.
2. Separate evidence from inference.
3. Compare options using visible criteria.
4. State uncertainty directly.
5. End with a recommendation or a clearly stated next research step.

## Default output order
1. Question
2. Context
3. Evidence summary
4. Option comparison
5. Risks / gaps
6. Recommendation
7. Confidence / uncertainty

## Quality standard
A reader should be able to understand:
- what decision is being considered
- what evidence exists
- what the trade-offs are
- what is recommended
- what remains uncertain
