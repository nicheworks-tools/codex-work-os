# Example Escalation

## Case summary
Users are intermittently failing to receive expected alert notifications after a recent configuration update.

## User impact
- Affected users may miss time-sensitive alerts
- Severity is moderate to high depending on workflow reliance
- Urgency is elevated because the issue affects trust in notification delivery

## Evidence / reproduction
- Multiple reports appeared after the latest configuration change
- Failures are intermittent rather than universal
- Current evidence suggests the issue may be tied to threshold or routing behavior, but this is not yet confirmed

## Actions already taken
- Support gathered examples from affected users
- Basic config review was completed
- No final root cause has been confirmed yet

## Escalation recommendation
Escalate to the product or engineering owner for focused review of recent configuration and delivery path behavior.

## Ready-to-send escalation note
