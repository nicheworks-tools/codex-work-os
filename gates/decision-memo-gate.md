# Decision Memo Gate

Use this gate before finalizing any recommendation or comparison memo.

## Pass conditions
- The decision question is explicit
- Evidence and assumptions are separated
- Trade-offs are visible
- A recommendation is present
- Remaining uncertainty is acknowledged

## Fail conditions
- Recommendation without evidence
- Evidence dump without conclusion
- No explicit trade-offs
- False certainty
- Missing decision owner or next step

## Minimum fix rule
If the memo fails, force this structure:
Question -> Evidence -> Options -> Trade-offs -> Recommendation -> Uncertainty
