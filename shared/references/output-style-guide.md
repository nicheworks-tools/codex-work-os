# Output Style Guide

## Priority order

Prefer outputs that are:

1. useful
2. clear
3. compact
4. reusable
5. polished

Polish matters, but not before usefulness.

---

## Tone guidance

Use a tone that is:

- calm
- direct
- practical
- non-performative

Avoid:

- exaggerated confidence
- marketing fluff
- ornamental verbosity
