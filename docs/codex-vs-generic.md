# Codex vs Generic Prompt Collections

## Why this repository is not just a generic prompt collection

A generic prompt collection usually gives you:

- broad prompts
- writing tips
- one-off output examples

This repository aims to go further.

It includes:

- pack-level `SKILL.md` files
- shared references and templates
- installation helpers
- repository validation scripts
- examples and source-of-truth specs

That means the goal is not only to inspire a better answer.
The goal is to make repeated Codex usage more consistent and portable.

---

## Why this repository is not just a copy of another ecosystem

Public skill repositories can be useful for learning, but this repository is intentionally shaped around its own structure and practical use model.

The aim is not to mimic another product’s public identity.
The aim is to build a coherent operating system that feels native to Codex.
