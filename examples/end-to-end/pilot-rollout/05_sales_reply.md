# Sales Reply

## Goal
Reply to the prospect without overcommitting while keeping the pilot conversation moving.

## Confirmed facts
- The prospect wants a lightweight pilot
- They want to start soon
- A smaller initial rollout is feasible in principle

## Open questions
- Number of pilot users
- Systems that must be included
- Any fixed launch deadline

## Recommended next step
Acknowledge feasibility, ask the minimum missing questions, and propose a short follow-up call.

## Ready-to-send draft
Thanks for reaching out. A lightweight pilot sounds like a good fit, and a smaller initial rollout is usually the fastest way to get started.

To help us recommend the best setup, could you share a bit more about the expected number of users, any systems that would need to be included, and whether you have a target timeline in mind?

Once we have that context, we can outline the best next step and what the pilot would likely involve. If helpful, we can also set up a short call to walk through it together.
